# AI Projects Portfolio

A React-based website showcasing machine learning and artificial intelligence projects for professional career growth. Built with Vite for fast development and optimized for deployment on Vercel.

## Features

- Showcase of 6 AI/ML projects
- Responsive design
- Clean, professional layout
- Technology tags for each project
- GitHub links for project repositories

## Projects Included

1. **Image Classification with CNN** - Deep learning for computer vision
2. **Chatbot with NLP** - Natural language processing and conversational AI
3. **Recommendation System** - Collaborative filtering for personalized suggestions
4. **Time Series Forecasting** - LSTM networks for predictive analytics
5. **Sentiment Analysis** - BERT-based text classification
6. **Anomaly Detection** - Unsupervised learning for IoT data

## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/WebApplication_vibecode.git
   cd WebApplication_vibecode
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## Deployment to Vercel

1. Push your code to a GitHub repository.

2. Connect your GitHub repository to Vercel:
   - Go to [vercel.com](https://vercel.com)
   - Sign in with your GitHub account
   - Click "New Project"
   - Import your repository

3. Vercel will automatically detect it as a Vite React project and deploy it.

4. Your site will be live at a Vercel-provided URL (e.g., `your-project.vercel.app`).

## Customization

- Edit `src/App.jsx` to modify the projects list
- Update styles in `src/App.css`
- Change the title in `index.html`
